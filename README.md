Bot-Plugin
----------

Made by: rkt_lvr  
  
### A Minecraft plugin that handles bots on your server.

This Plugin is meant to handle bots you make for your server. This project is meant  
to be used with the Python library to add logic to your bots. This project comes with the  
MIT license and has no warranty of any kind and should be used at your own risk.  
This project is a hobby project and should not be expected to be updated too much and  
be stable. Feel free to fork this project and add your own things. The inspiration for this project was mineflayer.

Happy Coding!
